---------------------------------О "PANZERCHASM"--------------------------------

"PanzerChasm" - свободное воссоздание игры "Chash - The Rift".
Для работы требуются данные оригинальной игры (файл CSM.BIN).


------------------------------СИСТЕМНЫЕ ТРЕБОВАНИЯ------------------------------

* Клавиатура, мышь.
* Операционная система Windows XP или более новая.
* Поддержка OpenGL 3.3 в системе/драйверах (для OpenGL режима).
* Не менее 128 Мб памяти.


----------------------------ИНСТРУКЦИЯ ПО ПРИМЕНЕНИЮ----------------------------

Поместить CSM.BIN от оригинальной игры в директорию с PanzerChasm.exe. Запустить
PanzerChasm.exe.

Можно указать другой файл, содержащий данные игры через параметр командной
строки --csm, например так:
	PanzerChasm.exe --csm CSM_RUS.BIN

Для запуска дополнения следует дополнительно указать путь к нему через параметр
командной строки --addon, например так:
	PanzerChasm.exe --addon ADDON1

Управление: WASD - передвижение, SPACE - прыжок, левая кнопка мыши - стрелять,
цифры 1-8 - выбор оружия, "~" - открыть/закрыть консоль, ESC - меню.
Часть управления можно изменить в меню настроек.

В консоли есть ряд команд. Чтобы увидеть их список, нунжо нажать TAB.
